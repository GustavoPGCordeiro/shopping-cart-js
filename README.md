# JavaScript Shopping Cart

## Description

This project is a simple shopping cart application developed with JavaScript. It allows users to select products, specify quantities, calculate the total purchase value, and manage cart contents dynamically.

The application updates the interface in real time, demonstrating how JavaScript can be used to manipulate HTML elements and create interactive web experiences.

## Features

### Product Selection

* Select products from a predefined list.
* Extract product name and unit price automatically.

### Quantity Management

* Allows users to choose the quantity of each product.
* Calculates the subtotal for each item.

### Shopping Cart

* Adds products dynamically to the cart.
* Displays selected products and quantities.

### Total Price Calculation

* Calculates the running total of all products added.
* Updates the total value automatically.

### Cart Reset

* Clears all products from the cart.
* Resets the total amount to zero.

### Dynamic Interface Updates

* Uses DOM manipulation to update cart information instantly.

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/GustavoPGCordeiro/shopping-cart-js.git
```

2. Open the project folder.

3. Open the `index.html` file in your browser.

4. Select products and quantities to simulate purchases.

## Core Functionalities

### Add Product

* Reads the selected product.
* Retrieves the quantity entered by the user.
* Calculates the subtotal.
* Updates the cart display.

### Calculate Total

* Maintains a running total of all products.
* Updates the displayed total automatically.

### Clear Cart

* Removes all items from the cart.
* Resets the total value.

## Example Workflow

```text
1. Select a product
2. Enter a quantity
3. Click "Add"
4. Product appears in the cart
5. Total amount is updated
6. Repeat as needed
7. Click "Clear" to reset the cart
```
